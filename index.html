222已讀不回助理問題排解

親愛的用戶您好，感謝您使用已讀不回助理，若您在使用上遇到困難，請先查看以下常見問題的解決方法，祝您使用愉快。

  • 問題排解

1.收不到訊息怎麼辦?

確認已開啟通知存取權，點我啟用

2.LINE的設定

在LINE設定中的「提醒」確認開啟以下功能
勾選✓「顯示訊息內容」
勾選✓「提醒」

3.Facebook的設定

在Messenger的設定中的「通知與音效」確認啟用以下功能
啟用✓「通知與音效」
啟用✓「通知預覽」

4.保持網路暢通

已讀不回助理只能截到通知欄所顯示的通知，若在您開啟網路功能時，同時進來很多訊息，但您的通訊軟體(LINE或Messenger)只顯示最後一則訊息，那就只能截到最後一則訊息，因此保持網路暢通，才能截到大部份的訊息，另外若手機處於省電模式也有可能在待機的時候被系統關掉了網路，而攔截不到訊息唷。

5.電池相關系統設定

同上述為了網路暢通時，才能順利地擷取訊息，因此請確認您的手機在省電相關的設定中排除已讀不回助理、LINE、Facebook以及其他您需要支援的通訊軟體。

6.使用一陣子後無法接收到訊息?

部分手機系統會自動為了效能，會不定期的清除其他App的背景服務，因此已讀不回助理也很有可能被清理掉，因此為了確保已讀不回助理的服務正常運作，當您使用一陣子後無法收到訊息時，請嘗試重新啟動手機。

7.有什麼App會影響已讀不回助理運作呢?

若您有安裝省電、防毒、效能優化等相關APP時，這些APP可能會為了加速系統運行，並將已讀不回助理的背景服務停止，請將已讀不回助理、LINE、Messenger設定為白名單。

8.為什麼群組訊息卻被歸類在個人訊息?

已讀不回助理只能截到通知欄所顯示的通知，您會發現LINE有時在通知欄顯示訊息時，並沒有顯示來自哪個群組，因此已讀不回助理也無法辨別訊息來自哪個群組，非常抱歉是我們做得不夠好，但LINE官方的通知設計若不改良，我們也無從修改，請您見諒。

9.為什麼LINE無法支援圖片、影片、錄音?

有鑑於Android完善系統的安全性，已讀不回助理無法讀取來自LINE、Messenger或其他通訊軟體中更隱密的訊息，對此我們也深感遺憾，請您見諒。

10.為什麼開啟LINE電腦版時已讀不回助理就收不到訊息了?

在電腦版的LINE中，預設會在開啟電腦版時關閉手機的通知，此時便會收不到訊息，若要確保已讀不回助理正常運作，請在電腦版的LINE中設定，「提醒」→「提醒規則」→取消勾選「使用電腦版時，智慧手機將不會收到提醒」。

11.為什麼Messenger的訊息不完整?

已讀不回助理只能截到通知欄所顯示的通知，若Messenger所能提供的最大長度訊息小於實際訊息就會發生此狀況，這是Messenger的運作機制，對此我們也深感遺憾，請您見諒。

12.我是OPPO用戶，遇到了些問題

親愛的OPPO用戶您好，因OPPO系統的特性，您會發現LINE與Messenger的通知，並不會每次都出現，因此也造成廣大OPPO用戶的困擾，但幸運的是我們提供了一些改善方法。教學以 OPPO R9s 為例，如果你使用的是其他 OPPO 機型請參考下面項目自行找尋對應設定哦，重新開機或是長時間待機後，請記得檢查設定有沒有跑掉喔。教學要開始了已讀不回助理、LINE、Messenger、Whatsapp與已讀不回助理有支援的通訊軟體，以下簡稱App們。

步驟一：進入手機的[安全中心] → [權限隱私] → [自動啟動管理] →把[App們] 設為背景啟動

步驟二：進入手機的[安全中心] → [權限隱私] → [懸浮視窗管理] → [App們]設為允許

步驟三：進入手機的[安全中心] → 點擊[一鍵體檢] → 等待掃描完成後→ 選擇[安全防護] → 點擊第一項(不是按關閉鍵) → 進入後把[App們]加入白名單

步驟四：點選 [設定] → [通知與狀態列] → [通知管理] → 開啟[允許通知]，允許App們

步驟五：點選 [設定] → [電池] → [耗電細節] → 選擇[App們] → 關閉[背景凍結]、[檢測到異常，自動最佳化]

步驟六：（確認軟體是否正常運作）：進入手機的[安全中心] → 選擇[自啟動應用程式白名單] → 裡面有[App們]則為已成功啟動
