既読回避サポーター setting on ASUS devices

Because of the varying optimization software used by different brands, 既読回避サポーター may not work normally on all devices. Please take note of the following instructions and set your device accordingly.

  • START

Reminder:

This set of instructions is suitable for ASUS Zenfone 2 only. If you are using another ASUS device, please find the corresponding settings in the other instructions below.

Step 1

Tap [Settings] → Tap [ Battery ] → Tap right top [ More ] → Tap [ Battery optimization ] → Tap [ All Apps ]→ Find [ 既読回避サポーター ] and select [Don't optimize]

Step 2

Go to [ Mobile Manager ] → [ Boost ] → [ Enable Super Boost ] → check [ 既読回避サポーター ]

（👉In order to prevent message APPs killed by system, please repeat above steps and apply on your message APPs(WhatsApp, Facebook, Instagram...),👈）
