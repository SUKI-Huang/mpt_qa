既読回避サポーター setting on SONY devices

Because of the varying optimization software used by different brands, 既読回避サポーター may not work normally on all devices. Please take note of the following instructions and set your device accordingly.

  • START

Reminder:

This set of instructions is suitable for SONY Z2 only. If you are using another SONY device, please find the corresponding settings in the other instructions below.

Step 1

Tap [Apps] → Tap [ Settings ] → Tap [ Battery ] → Tap right top menu icon [ Battery optimization ] → Find [ 既読回避サポーター ] and check. it
