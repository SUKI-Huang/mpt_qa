Huawei 手機 已讀不回助理 設定，適用於Mate 8與其他Huawei裝置

由於各廠牌使用的不同 [客製效能優化] 或 [省電] 軟體，可能因此會導致 已讀不回助理 無法正常運作，大家只要按照教學逐步設定就可以順暢使用 已讀不回助理 囉。請使用 Huawei 手機的 已讀不回助理 用戶嘗試以下步驟設定，現在就跟著一起操作看看吧！

  • 開始步驟

小提醒：

（1）此教學圖片來自 Huawei Mate 8 機型，如果你使用的是其他 Huawei 機型請參考下面項目自行找尋對應設定哦

（2）重新開機或是長時間待機後，請記得檢查設定有沒有跑掉喔

步驟一

點選 [設定] → [電池] → 選擇 右上角[齒輪] → [休眠狀態時連線] → 休眠時仍保持連線

步驟二

[設定] → [應用程式] → 選擇 [已讀不回助理] → 選擇 [通知] 並設定如下圖所示

步驟三

[設定] → [應用程式] → 選擇 [已讀不回助理] → 選擇 [電池] 並設定如下圖所示

步驟四

點下 [多工鍵] → 把 [已讀不回助理] 鎖上

（鎖頭出來才表示 已讀不回助理 不會被 Huawei 的系統清除喔）

步驟五

[手機管理員] → [電池] → 選擇 [在畫面鎖定後關閉應用程式] → 將 [已讀不回助理] 設定為不要關閉

步驟六

[手機管理員] → [自動啟動] → 將 [已讀不回助理] 設定為隨系統啟動

步驟七

[設定] → [應用程式] → [設定] → [特別存取權] → [忽略電池最佳化] → 將 [已讀不回助理] 設定為允許

步驟八

[設定] → [應用程式] → [設定] → [特別存取權] → [通知存取權] 將 [已讀不回助理] 開啟

（👉記得重複以上步驟，設定您的訊息APP(Facebook, LINE, Instagram...)，已免訊息APP被系統清除，影響已讀不回助理的運作唷👈）
