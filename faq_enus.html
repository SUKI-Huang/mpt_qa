Questions and answers

Thank you for using Message Peeping Tom(MPT), if you are facing some problems when you use MPT, please check the Q & A first.

1.Can not receive message?

Ensure the 「Notification Access」 feature was enabled, Click me to enable.

2.Facebook Messenger settings

Open the setting page in Facebook Messenger, and go「Notifications & Sounds」checks the feature down below.
Enable ✓「Notifications & Sounds」
Enable ✓「Notification Previews」

3.LINE settings

Open the setting page in LINE, and go「Notifications」checks the feature down below.
Enable ✓「Notifications」
Enable ✓「Message previews」

4.Why LINE always show 「You have new messages」?

Open the setting page in LINE, and go「Chats & calls」cancel the feature down below.
Cancel 「Letter Sealing」

5.Keep network available

MPT can only catch messages from notification bar, if your deivce receive too much messages at same time when you turning on the network, at that time your Message Apps(Messenger, Whatsapp...) only show the latest one on the notificaton bar, then the MPT will only catch the latest one, so please ensure that keeping network available to receive messages perfectly.

Device network feature might turned off by system, if device is on battery save mode, please add MPT and message Apps to the whitelist.

6.Battery-related settings

In order to keep the network available and receive messages normally, please add MPT and message Apps to the whitelist on battery save mode, that can avoid system kill MPT service and message Apps services.

7.MPT can not receive messages after few days?

Some android devices will kill Apps background services automatically to enhace system performance, so MPT and other message Apps could be killed sometimes, in order to ensure that the MPT work perfectly, please try to restart your device.

8.What Apps might stop MPT working?

If there are Apps about battery save,anti-virus, enhance performance or memory cleaner in your device, it might stop kill/stop MPT service, so please ensure that add MPT and other message Apps to the whitelist.

9.Why did MPT receive group message as personal message ?

MPT can only catch messages from notification bar, you will find that sometimes message Apps only show the person's name instead of group name, so that MPT can not put the message to the correct group but the personal message, we apologize for that and thank you for your understanding.

10.Why MPT can not support image, video and voice?

According to Android system security, MPT can not catch image, video and voice from other message Apps, we apologize for that and thank you for your understanding.

11.MPT can not work when I'm using Messenger on PC or Mac?

LINE

If you are using LINE on PC/MAC, it will close notification on your mobile device automatically, and then MPT will not receive message, in order to work normally please do some step on your PC/Mac verion,「Setting」→「Notifications」→「Notification Rules」→uncheck「Turn off smartphone notifications when logged in to the PC version」

Facebook Messenger

If you are using Messenger on Website, Facebook will turn off notification on your mobile device automatically.

12.Why the message is incomplete?

MPT can only catch messages from notification bar, if the notification message was cut by message Apps's, MPT can only catch the incomplete message, we apologize for that and thank you for your understanding.

13.I am OPPO user, I am faceing some problems

Dear OPPO users, according to OPPO system design, you will find that notification not always shows on notification bar, it affects many users, fortunately we provide a solution to solve it. Example for R9s, Other OPPO devices are similar way to solve it. MPT, Messenger, LINE and WhtatsApp we call Apps down below.

Step 1：Oepn [ Security Center ] → [ Privacy permission ] → [ auto-run management ] →enable [ Apps ]

Step 2：Oepn [ Security Center ] → [ Floating windows management ] → enable [ Apps ]

Step 3：Oepn [ Security Center ] → [ Quick scan ] → after scanned → click [ Security Protection ] → Find [ Apps ] and Add to whitelist

Step 4 : Open [ Setting ] → [ Notification and status Bar ] → [ Manage notifications ] → Find [ Whoscall ] and setting like below

Step 5 : Open system [ Setting ] → [ Battery ] → find [ Apps ] and uncheck [ Freeze when in background ] and [ Automatically optimize when an abnormally is detected]

Step 6：(If you want to check Apps is enable) :Open [ Setting ] → [ Security Center ] → [ Startup app whitelist ] → Find [ Apps ]
