已读不回助理问题排解

亲爱的用户您好，感谢您使用已读不回助理，若您在使用上遇到困难，请先查看以下常见问题的解决方法，祝您使用愉快。

  • 问题排解

1.收不到信息怎麽办?

确认已开启通知访问，点我启用

2.LINE的设置

在LINE设置中的「提醒」确认开启以下功能
勾选✓「显示信息内容」
勾选✓「提醒」

3.为什麽LINE总是收到「您有新信息」?

在LINE设定中的「聊天与通话」关闭以下功能
取消「Letter Sealing」

4.Facebook的设置

在Messenger的设置中的「通知与音效」确认启用以下功能
启用✓「通知与音效」
启用✓「通知预览」

5.保持网路畅通

已读不回助理只能截到通知栏所显示的通知，若在您开启网路功能时，同时进来很多信息，但您的通讯软体(LINE或Messenger)只显示最后一则信息，那就只能截到最后一则信息，因此保持网路畅通，才能截到大部份的信息，另外若手机处于省电模式也有可能在待机的时候被系统关掉了网路，而拦截不到信息唷。

6.电池相关系统设置

同上述为了网路畅通时，才能顺利地撷取信息，因此请确认您的手机在省电相关的设置中排除已读不回助理、LINE、Facebook以及其他您需要支援的通讯软体。

7.使用一阵子后无法接收到信息?

部分手机系统会自动为了效能，会不定期的清除其他App的背景服务，因此已读不回助理也很有可能被清理掉，因此为了确保已读不回助理的服务正常运作，当您使用一阵子后无法收到信息时，请尝试重新启动手机。

8.有什麽App会影响已读不回助理运作呢?

若您有安装省电、防毒、效能优化等相关APP时，这些APP可能会为了加速系统运行，并将已读不回助理的背景服务停止，请将已读不回助理、LINE、Messenger设置为白名单。

9.为什麽群组信息却被归类在个人信息?

已读不回助理只能截到通知栏所显示的通知，您会发现LINE有时在通知栏显示信息时，并没有显示来自哪个群组，因此已读不回助理也无法辨别信息来自哪个群组，非常抱歉是我们做得不够好，但LINE官方的通知设计若不改良，我们也无从修改，请您见谅。

10.为什麽LINE无法支援图片、影片、录音?

有鑑于Android完善系统的安全性，已读不回助理无法读取来自LINE、Messenger或其他通讯软体中更隐密的信息，对此我们也深感遗憾，请您见谅。

11.为什麽开启LINE电脑版时已读不回助理就收不到信息了?

在电脑版的LINE中，预设会在开启电脑版时关闭手机的通知，此时便会收不到信息，若要确保已读不回助理正常运作，请在电脑版的LINE中设置，「提醒」→「提醒规则」→取消勾选「使用电脑版时，智慧手机将不会收到提醒」。

12.为什麽Messenger的信息不完整?

已读不回助理只能截到通知栏所显示的通知，若Messenger所能提供的最大长度信息小于实际信息就会发生此状况，这是Messenger的运作机制，对此我们也深感遗憾，请您见谅。

13.我是OPPO用户，遇到了些问题

亲爱的OPPO用户您好，因OPPO系统的特性，您会发现LINE与Messenger的通知，并不会每次都出现，因此也造成广大OPPO用户的困扰，但幸运的是我们提供了一些改善方法。教学以 OPPO R9s 为例，如果你使用的是其他 OPPO 机型请参考下面项目自行找寻对应设置哦，重新开机或是长时间待机后，请记得检查设置有没有跑掉喔。教学要开始了已读不回助理、LINE、Messenger、Whatsapp与已读不回助理有支援的通讯软体，以下简称App们。

步骤一：进入手机的[安全中心] → [权限隐私] → [自动启动管理] →把[App们] 设为背景启动

步骤二：进入手机的[安全中心] → [权限隐私] → [悬浮视窗管理] → [App们]设为允许

步骤三：进入手机的[安全中心] → 点击[一键体检] → 等待扫描完成后→ 选择[安全防护] → 点击第一项(不是按关闭键) → 进入后把[App们]加入白名单

步骤四：点选 [设置] → [通知与状态列] → [通知管理] → 开启[允许通知]，允许App们

步骤五：点选 [设置] → [电池] → [耗电细节] → 选择[App们] → 关闭[背景冻结]、[检测到异常，自动最佳化]

步骤六：（确认软体是否正常运作）：进入手机的[安全中心] → 选择[自启动应用程式白名单] → 裡面有[App们]则为已成功启动
